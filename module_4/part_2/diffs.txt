Тесты сгенерированные в IDE:
 - Просто создаются
 - Не поддерживают параметризацию
 - Нет assert - тест просто упадет, каждое падение нужно будет разбирать отдельно
 - Невозможно использовать данные "снаружи" тестов (например, забирать из базы тестовые креды)
 - Тесты не могут быть запущены на чистом окружении или в разных браузерах (используется только текущий браузер с текущей сессией)
 - Куча предопределенных импортов, которые не используются
 - Без изменения кода невозможно поменять вводимые данные
 - Из-за локаторов разбросанных по коду - будет сложно поддерживать
